В .example.env
TELEGRAM_BACKEND_HOST=бэкенд телеграм бота
CHATID=Чат Id пользователя которому будут приходить сообщения
PARTNER_API = Это хост для промокодов
X_API_KEY=SOME_UR_KEY
CDN_URL=URL вашего CDN (например https://cdn.smartcardio.ru). Используется как assetPrefix в next.config.js - все статические ассеты (JS, CSS бандлы) будут загружаться через указанный CDN. Если не задано, ассеты грузятся с основного домена как обычно.
